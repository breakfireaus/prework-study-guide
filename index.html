CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • We can style HTML elements using the elements as selectors or by using the class attribute as a selector.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • When a developer wants to style an element directly in the HTML file, they use inline CSS.
  • It is also possible to embed an entire stylesheet directly in the HTML file using a style element in the head element.
  • External CSS style sheet is preferred in most cases because it allows developers to keep all their CSS rules in a separate file.
  • we can style HTML elements using the elements as selectors or by using the class attribute as a selector.

Git

  • When creating a repository, make sure you create with a readme and licence.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • Branches are created not to damage the main branch.
  • Developers PULL the main branch in local repositories.
  • When developers finish coding they PUSH it to the main branch for peers to review.

JavaScript

  • JavaScript is a programming language that allows users to interact with websites.
  • One core feature is variables, which store data types as values.
  • Control flow is the order in which a computer executes code in a script.
  • using conditional statements such as if statements allows us to interrupt control flow.
  • When creating an if statement, we provide one or more conditions for the computer to evaluate.
